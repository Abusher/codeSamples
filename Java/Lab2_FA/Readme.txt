Программа для определения еквивалентности двух автоматов, которые подаются на вход в файлах.
Программа сначала минимизирует каждый из автоматов а потом определяет их еквивалентность.

формат файла автомата:

1. Кол-во символов в алфавите
2. Кол-во состояний автомата
3. Начальное состояние
4. Кол-во финальных состояний и их номера
5. Строки переходов
